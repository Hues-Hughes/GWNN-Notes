
		**Virtualization**
		
			Virtual  Machines and HyperVisors
		
				HyperVisor - runs multiple independent Operating Systems on 1 physical system
				
					Type 1
						Hardware virtualization - guest OS runs directly on hardware
						
					Type 2
						Hosted virtualization - hypervisor runs on top of host OS
			
			
			Containers - Barebones no guest OS, just the app and dependcies. Docker delivers software in containers, K8S orchestrates and manages them.
			
			Virtual Desktop Infrastructure (VDI) hosts desktop envs on a central server
